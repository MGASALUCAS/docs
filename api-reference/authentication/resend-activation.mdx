---
title: 'Resend Activation'
api: 'POST /api/auth/resend-activation/'
---

Resend activation code to user's email or phone.

<ParamField body="email" type="string" required>
  User email address
</ParamField>

## Request Example

<RequestExample>

```json
{
  "email": "user@example.com"
}
```

</RequestExample>

## Success Response (200)

<ResponseField name="message" type="string">
  Success message
</ResponseField>

<ResponseField name="email_sent" type="boolean">
  Whether email was sent
</ResponseField>

<ResponseField name="sms_sent" type="boolean">
  Whether SMS was sent
</ResponseField>

<ResponseExample>

```json
{
  "message": "Activation code has been sent to your email/phone",
  "email_sent": true,
  "sms_sent": false
}
```

</ResponseExample>
