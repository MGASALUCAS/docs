---
title: 'Login'
api: 'POST /api/auth/login/'
---

Authenticate and receive access tokens.

<ParamField body="email" type="string" required>
  User email address
</ParamField>

<ParamField body="password" type="string" required>
  User password
</ParamField>

## Request Example

<RequestExample>

```json
{
  "email": "user@example.com",
  "password": "SecurePassword123!"
}
```

</RequestExample>

## Success Response (200)

<ResponseField name="message" type="string">
  Success message
</ResponseField>

<ResponseField name="user" type="object">
  User object with profile details
</ResponseField>

<ResponseField name="tokens" type="object">
  Access and refresh tokens
</ResponseField>

## Error Response (400)

<ResponseExample>

```json
{
  "non_field_errors": ["Invalid credentials."]
}
```

</ResponseExample>

## Error Response (400) - Account Not Verified

<ResponseExample>

```json
{
  "non_field_errors": [
    "Your account has not been activated. We have sent a 6-digit verification code to your email. Please check your email and use the code to verify your account, or visit /api/auth/resend-activation/ to resend."
  ]

```

</ResponseExample>
