---
title: 'Get Unread Count'
api: 'GET /api/notifications/unread-count/'
---

Get count of unread notifications.

## Success Response (200)

<ResponseField name="unread_count" type="integer">
  Number of unread notifications
</ResponseField>

<ResponseExample>

```json
{
  "unread_count": 5
}
```

</ResponseExample>
