---
title: 'Create Template'
api: 'POST /api/messaging/templates/'
---

Create a new SMS message template.

<ParamField body="name" type="string" required>
  Template name
</ParamField>

<ParamField body="category" type="string" required>
  Template category (e.g., TRANSACTIONAL, PROMOTIONAL)
</ParamField>

<ParamField body="language" type="string" required>
  Template language code (e.g., en)
</ParamField>

<ParamField body="message" type="string" required>
  Template message with variables (e.g., {{name}})
</ParamField>

<ParamField body="variables" type="array">
  Array of variable names used in the template
</ParamField>

## Request Example

<RequestExample>

```json
{
  "name": "Welcome Message",
  "category": "TRANSACTIONAL",
  "language": "en",
  "message": "Welcome {{name}}! Thank you for joining us.",
  "variables": ["name"]
}
```

</RequestExample>
