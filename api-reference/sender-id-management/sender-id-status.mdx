Get approval status of a sender ID.

<ParamField path="sender_id" type="string" required>
  Sender ID ID
</ParamField>

<ResponseExample>

```json
{

  "sender_id": "MYBIZ",
  "status": "approved",
  "approved_at": "2025-11-28T10:00:00Z",
  "is_default": false
}
```

</ResponseExample>
